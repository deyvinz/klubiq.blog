# Klubiq.Blog

A modern blog application for Klubiq Property Management SaaS company, built with Next.js, TypeScript, and Tailwind CSS.

## Project Information

- **Name**: Klubiq.Blog
- **Title**: Klubiq Property Management SaaS Blog
- **Author**: Glumia Solutions
- **Version**: 0.1.0

## Tech Stack

- **Frontend**: Next.js 14, React 18, TypeScript
- **Styling**: Tailwind CSS
- **Fonts**: Inter (headings), Open Sans (body)
- **CMS**: Strapi (self-hosted)
- **Database**: PostgreSQL

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure
